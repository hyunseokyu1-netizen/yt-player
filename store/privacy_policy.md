**ChainPlay**
최종 업데이트: 2026년 5월

## 수집하는 정보

ChainPlay는 사용자의 개인정보를 수집하지 않습니다.

앱이 기기에 저장하는 정보:
- 사용자가 직접 추가한 YouTube 영상 URL 및 제목, 썸네일 (기기 로컬 저장소에만 저장)

이 데이터는 외부 서버로 전송되지 않으며, 앱 삭제 시 함께 삭제됩니다.

---

## 네트워크 통신

앱은 다음 목적으로만 인터넷을 사용합니다:

1. **영상 정보 조회**: URL 추가 시 YouTube oEmbed API (youtube.com/oembed)를 호출하여 영상 제목과 썸네일을 가져옵니다.
2. **영상 재생**: YouTube 공식 플레이어(WebView)를 통해 영상을 스트리밍합니다.

위 통신은 모두 YouTube/Google의 서버와 직접 이루어지며, 개발자의 별도 서버를 거치지 않습니다.

## Privacy Policy (English)

**ChainPlay** does not collect any personal information from users.

The app stores only YouTube video URLs, titles, and thumbnails that the user explicitly adds — solely on the device's local storage. This data is never transmitted to any external server and is deleted when the app is uninstalled.

The app accesses the internet only to:
1. Fetch video info via YouTube's oEmbed API when a URL is added
2. Stream videos through the official YouTube player (WebView)

All network communication occurs directly with YouTube/Google servers. No developer-operated server is involved.

This app uses YouTube (Google LLC) services. Google's Privacy Policy applies to that usage: https://policies.google.com/privacy
