---

## 전체 설명 (4000자 이내)

유튜브 영상을 나만의 플레이리스트로 묶어 순서대로 자동 재생하세요.

**주요 기능**

• URL 붙여넣기만으로 영상 추가 — youtube.com, youtu.be, Shorts 모두 지원
• 자동 순차 재생 — 한 영상이 끝나면 다음 영상으로 자동 이동
• 이전 / 다음 버튼으로 빠른 탐색
• 플레이리스트 순서 자유롭게 변경
• 목록 자동 저장 — 앱을 껐다 켜도 그대로
• 영상 제목과 썸네일 자동 표시

**이런 분께 추천**

- 같은 유튜브 영상을 반복해서 보시는 분
- 특정 영상들을 정해진 순서로 이어 보고 싶은 분
- 유튜브 앱의 복잡한 기능 없이 간단한 재생기가 필요한 분

**지원 URL 형식**

- youtube.com/watch?v=...
- youtu.be/...
- youtube.com/shorts/...
- youtube.com/embed/...

이 앱은 개인 사용 목적으로 만들어진 간단한 YouTube 플레이리스트 플레이어입니다.
재생은 YouTube의 공식 플레이어를 통해 이루어지며, 별도의 계정 로그인이 필요 없습니다.

## Full Description

Build your own YouTube playlist and watch videos one after another — automatically.

**Key Features**

• Add videos by pasting any YouTube URL — supports youtube.com, youtu.be, and Shorts
• Auto-play next — when one video ends, the next one starts automatically
• Previous / Next buttons for quick navigation
• Reorder your playlist freely
• Playlist saved automatically — picks up right where you left off
• Video title and thumbnail loaded automatically

**Perfect for**

- Watching the same set of YouTube videos repeatedly
- Playing specific videos in a fixed order without distractions
- Anyone who wants a simple, no-frills YouTube player

**Supported URL Formats**

- youtube.com/watch?v=...
- youtu.be/...
- youtube.com/shorts/...
- youtube.com/embed/...

This is a lightweight personal YouTube playlist player. Playback is handled entirely through YouTube's official player — no account login required.
